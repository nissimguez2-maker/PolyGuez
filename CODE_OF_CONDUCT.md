# Code of Conduct

We are committed to providing a friendly, safe, and welcoming environment for everyone.

## Our standards
- Be respectful and constructive.
- Assume good intent.
- No harassment, discrimination, or hate speech.
- Keep discussions professional and on-topic.

## Reporting
If you experience or witness unacceptable behavior, please open a GitHub issue or contact the maintainers privately.

## Enforcement
Project maintainers may remove, edit, or reject contributions that violate this Code of Conduct.

This project follows the Contributor Covenant spirit.
