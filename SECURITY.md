# Security Policy

## Reporting a Vulnerability

We take the security of this project seriously.

If you discover a security vulnerability, please do **not** report it through public GitHub issues. Instead, please report it privately.

You can disclose the vulnerability by sending an email to the repository owners or creating a draft security advisory if you have permissions.

Please include:
- A description of the vulnerability.
- Steps to reproduce.
- Potential impact.

We will do our best to respond and fix the issue as soon as possible.
