# Security Policy

We take the security of the Polymarket Agents project seriously and appreciate responsible disclosures.

## Supported versions

Only the latest version of the `main` branch is actively supported. Older commits and forks may not receive security updates.

## Reporting a vulnerability

If you believe you have found a security issue in this repository:

- Do **not** open a public GitHub issue.
- Send an email with details and reproduction steps to **security@polymarket.com**.
- Include any relevant logs, environment information, and a minimal proof of concept if possible.

We will review your report, investigate the issue, and respond with next steps as soon as we can.
